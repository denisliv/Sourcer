# AlfaHRSourcer

Сервис для HR-специалистов: поиск кандидатов через API HeadHunter (rabota.by) с удобным веб-интерфейсом.

## Возможности

- Поиск резюме по названию позиции (в заголовке, навыках, позициях)
- Исключение нежелательных слов из результатов
- Фильтрация по опыту (без опыта, 1–3, 3–6, 6+ лет)
- Выбор периода обновления резюме (1–365 дней)
- Настройка количества результатов (до 2000)
- Таблица кандидатов с фото, именем, должностью, локацией, опытом, зарплатой и ссылкой
- Экспорт результатов в CSV

## Стек

- **Backend:** Python, FastAPI, httpx (async)
- **Frontend:** HTML, CSS, JavaScript
- **API:** HeadHunter (hh.ru / rabota.by)

## Установка и запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Указать токен доступа в `config.yaml`:

```yaml
ACCESS_TOKEN: "ваш_токен"
```

3. Запустить сервер:

```bash
python main.py
```

4. Открыть в браузере: **http://127.0.0.1:8000**
